# Contributing Guidelines

Thanks for contributing! Here's how:

1. **Question Format:**
````markdown
   ### Q#: Question title
   
   **Answer:**
   Explanation here
   
   **Code Example:**
```language
   code here
```
````

2. **Categories:** Place questions in correct sections
3. **Quality:** Clear, accurate, with examples
4. **No duplicates:** Search existing questions first

## Quick Start
1. Fork the repo
2. Add your question
3. Submit PR
4. Wait for review

Questions? Open an issue!
